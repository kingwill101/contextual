Contextual

info

Contextual is a powerful, flexible logging system for Dart that provides typed configuration, structured logging, multiple output channels, and ergonomic APIs.

Installation‚Äã

  • pubspec.yaml
  • Terminal
dependencies:
  contextual: ^2.0.0
dart pub add contextual

Quick start‚Äã

Get started with Contextual in just a few lines:

import 'package:contextual/contextual.dart';

void main() async {
  final logger = await Logger.create(
    config: LogConfig(
      channels: [ConsoleChannel(ConsoleOptions(), name: 'console')],
    ),
  );

  logger.info('Hello from Contextual!');
}
tip

That's it! Your logger is now ready to use with console output, pretty formatting, and structured logging capabilities.
